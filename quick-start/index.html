快速开始

通过下面的指令，快速初始化一个 TypeScript + React 组件项目。

npm init @ice/pkg react-component

# Or pnpm
pnpm init @ice/pkg react-component

启动项目：

cd react-component
npm run start

现在，访问 http://localhost:4444，尽兴地玩耍吧。
